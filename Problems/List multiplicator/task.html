List multiplicator Open on JetBrains Academy

You are provided with the backbone of ListMultiplicator class that has multiple methods that take a list and multiply its content by a specified number of times. The method should change the original list content and return the modified list.

Your task is to add implementation to the method without changing its signature.

It is guaranteed that:

  • the list is not null
  • n equals or greater than zero

n stands for the number of times the original list repeated in the result. In case n equals zero, the resulting list should be empty.

For example, if n equals 2 and the original list is:

[1, 2, 3]

the result should be:

[1, 2, 3, 1, 2, 3]

ListMultiplicator class

Well done!
